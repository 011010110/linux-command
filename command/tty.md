tty
===

显示连接到当前标准输入的终端设备文件名。

## 补充说明

**grename命令** 可以重命名卷组的名称。

### 语法  

```
vgrename [选项] [旧卷组路径|旧卷组名称|旧卷组UUID] [新卷组路径|新卷组名称]
```

### 选项  

```
-s 什么也不显示，只返回退出状态值
```

### 例子

显示连接到当前标准输入的终端设备文件名。

```bash
[root@localhost ~]# tty
/dev/pts/0
```
